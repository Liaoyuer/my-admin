# my-admin
一个基于vue-cli的后台管理框架，集成有百度富文本编辑器
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
